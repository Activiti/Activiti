Configure Dev Environment (OS X with HomeBrew installed)
=========================================================

* Update HomeBrew: 'brew update'

* Upgrade all HomeBrew packages: 'brew upgrade' (all packages! Use 'brew upgrade node' to upgrade node only

* (If not yet installed) install node.js: 'brew install node'

* Update of all NPM packages: 'npm -g update'

* Install Yeoman, Grunt and Bower: 'npm install -g yo grunt-cli bower' 

* Install angular/karma plugins for Yeoman: 'npm install -g generator-angular generator-karma'

* in src/main/webapp, execute 'npm install && bower install' to install local dependencies
